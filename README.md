# DBproject
## Онлайн биржа игровых товаров.

### Описание проекта

#### > Данная база данных является моей первой разработанной базой данных и предназначена для онлайн-биржи игровых товаров и игр.
*Проект создан в учебных целях для освоения принципов проектирования реляционных баз данных, работы с таблицами, связями и запросами*.
Онлайн-биржа позволяет пользователям **покупать**, **продавать** и **обменивать** игровые товары, такие как внутриигровые предметы, аккаунты и цифровые копии игр.

#### > Цель проекта
> - изучение структуры реляционной базы данных;
> - освоение связей между таблицами (один-к-одному, один-ко-многим);
> - получение практического опыта работы с SQL-запросами;
> - моделирование реальной системы онлайн-площадки для торговли цифровыми товарами.

#### > Основные сущности базы данных
В базе данных реализованы следующие основные сущности:
> - Пользователи (user) — информация о зарегистрированных пользователях биржи;
> - Лоты (lot) — объявления с названием описанием товаров;
> - Продукты (product) - сами товары в лотах;
> - Категории (category) - категории товаров;
> - Заказы (order) - купленные лоты;
> - Рейтинг заказа (rate) - отзыв о заказе и кол-во звёзд;
> - Агенты поддержки (support) - информация о агентах поддержки;

#### > Функциональные возможности

База данных поддерживает:
> - хранение данных о пользователях и их товарах;
> - размещение объявлений о продаже игровых предметов;
> - учёт заказов;
> - контроль статуса товаров (в продаже, снят с продажи);
> - формирование выборок и отчётов с помощью SQL-запросов.

#### > Используемые технологии

СУБД: **PostgreSQL**
Язык запросов: **SQL**
Тип базы данных: **реляционная**

#### > Структура проекта
Проект включает:
> - таблицы базы данных;
> - первичные и внешние ключи;

##### > Статус проекта
> - Проект находится на стадии учебной разработки и может быть расширен дополнительными функциями, такими как рейтинги пользователей, система отзывов и история операций.
